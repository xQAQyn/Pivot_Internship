# 选择器

## 选择器列表

- 用`,`分隔多个选择器，则规则同时对多个选择器生效
- **注意**：若选择器列表出现非法选择器，则规则对所有选择器均不生效

## 选择器的种类

- 类型选择器`h1{}`

- 类选择器`.box{}`

- id选择器`#box`

- 标签属性选择器

  - 根据元素上某个标签的属性是否存在进行选择：`a[title]{}`
  - 根据元素上某个标签的属性值是否为特殊值进行选择：`a[href="https://example.com"] { }`

- 全局选择器`*`

  - 目的增强可读性
  - 例如：`article :first-child {}`易与`article:first-child {}`混淆，则写作

  ```css
  article *:first-child{}
  ```
